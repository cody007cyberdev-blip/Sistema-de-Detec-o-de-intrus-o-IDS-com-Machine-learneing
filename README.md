# 🛡️ IDS Machine Learning
Sistema avançado de detecção de intrusão que utiliza modelos de aprendizagem automática para analisar o tráfego de rede e identificar comportamentos maliciosos em tempo real.

## ✨ Funcionalidades
- **Captura de Pacotes:** Monitorização ativa da interface de rede.
- **Deteção de Ataques:** Identificação de padrões de intrusão conhecidos e anomalias.
- **Visualização:** Gráficos e relatórios sobre a saúde da rede e ameaças detetadas.

## 🛠️ Tecnologias
- **Linguagem:** Python
- **ML:** Scikit-learn / TensorFlow
- **Análise:** Scapy + Pandas
